# 🎯 Focus — Sistema de Produtividade e Organização

## 📌 Sobre o Projeto

O Focus é uma plataforma moderna de produtividade desenvolvida para ajudar usuários a organizarem tarefas, metas e rotinas de forma simples, intuitiva e eficiente.
O projeto foi criado com foco em desempenho, experiência do usuário e design moderno, trazendo uma interface agradável e funcional inspirada em aplicações profissionais de organização e gerenciamento de tarefas.

A proposta principal do Focus é transformar produtividade em algo visual, leve e acessível, permitindo que qualquer pessoa consiga acompanhar seu progresso diário sem complicações.

# 🎨 Design do Projeto

O visual do projeto foi desenvolvido utilizando conceitos modernos de UI/UX, buscando uma experiência limpa e intuitiva.
As animações e efeitos de scroll deixam a navegação mais fluida e agradável, enquanto o uso de cards melhora a visualização das informações.

A identidade visual foi criada pensando em produtividade, foco e simplicidade.
